# Lab 6 #

This project demonstrates the implementation of fundamental data structures and algorithms using Python. It includes searching techniques, sorting algorithms, and basic linear data structures such as stacks and queues. The aim is to understand how different approaches affect program efficiency and performance.

Searching Techniques:

Searching algorithms are used to find the location of an element in a dataset.

Linear Search checks each element one by one and works on both sorted and unsorted data.
Indexed (Block) Search works on sorted lists by dividing data into blocks, making the search faster by reducing comparisons.

Sorting Algorithms:

Sorting arranges data in a specific order (ascending or descending).

Selection Sort repeatedly finds the smallest element from the unsorted part and places it at the correct position.
Although simple to implement, it is not efficient for large datasets due to its O(n²) time complexity.

Stack Data Structure:

A stack follows the LIFO (Last In, First Out) principle.
Elements are inserted using push() and removed using pop(). It is widely used in expression evaluation, recursion, and backtracking problems.

Queue Data Structure

A queue follows the FIFO (First In, First Out) principle.
Elements are inserted using enqueue() and removed using dequeue(). It is commonly used in scheduling and real-world waiting systems.

Recursion vs Iteration:

Recursion solves problems by calling the function itself and is useful for problems like factorial and Fibonacci series.
Iteration uses loops and is generally more efficient in terms of memory and execution speed.
